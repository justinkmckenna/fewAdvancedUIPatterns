Starter Project

To save a bit of time, I created a starter project.

This project is Angular 11.2.11, with TSLint replaced with ESLint.

To do this when you create a project, or to upgrade a project, see https://github.com/angular-eslint/angular-eslint

It also has a .vscode directory in the project root. This has some basic Angular settings, and the recommended extensions.

You can copy this into any other project we create.

Let's Rock.
